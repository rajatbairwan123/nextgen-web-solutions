# NextGen Web Solutions

NextGen Web Solutions is a business website developed using Laravel. This project includes a dynamic admin panel to manage services, portfolio, and contact leads.

## 🚀 Features

- Dynamic Services Management
- Portfolio Management
- Contact Form with Lead Management
- Admin Dashboard
- Authentication System
- Responsive UI

## 🛠️ Technologies Used

- Laravel
- PHP
- MySQL
- Bootstrap
- jQuery
- HTML/CSS

## 📁 Project Modules

- Admin Panel
- Services Management
- Portfolio Management
- Contact Lead Management
- Dashboard Analytics

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/rajatbairwan123/nextgen-web-solutions.git

2. Go to project directory
cd nextgen-web-solutions

3. Install dependencies
composer install

4. Copy .env file
cp .env.example .env

5. Generate key
php artisan key:generate

6. Setup database and run migration
php artisan migrate

7. Run project
